# ⚠️ Διορθώσεις Συνταγών σύμφωνα με Παιδιατρικές Οδηγίες (WHO, NHS, AAP)

Αυτό το αρχείο περιέχει τις απαραίτητες αλλαγές που πρέπει να γίνουν στο `index.html` για να είναι 100% σύμφωνες με τις διεθνείς παιδιατρικές οδηγίες.

## 📊 Συνοπτικά

Οι συνταγές σας είναι **90% σωστές**, αλλά υπάρχουν **3 μικρές διορθώσεις** που χρειάζονται για να γίνει 100% ασφαλές και σύμφωνο με τις οδηγίες:

- **WHO** (World Health Organization)
- **NHS** (UK National Health Service)
- **AAP** (American Academy of Pediatrics)

## 🔧 Αλλαγή #1: Μεταφορά Φάβας με Κολοκύθα (Συνταγή #21)

### Πρόβλημα
Η **Φάβα με Κολοκύθα** είναι στο **Στάδιο 2 (7-9 μήνες)** αλλά έχει badge **"8+ μήνες"**. Η φάβα είναι όσπριο με υψηλές φυτικές ίνες και μπορεί να είναι δύσκολη για το πεπτικό σύστημα μωρών 7-8 μηνών.

### Λύση
Μεταφέρετε τη συνταγή στο **Στάδιο 3 (9-12 μήνες)** και αλλάξτε το badge σε **"9+ μήνες"**.

### Αναλυτικά Βήματα

1. **Βρείτε** τη συνταγή #21 στο HTML (είναι στο Στάδιο 2):
   ```html
   <!-- Συνταγή 21 -->
   <article class="recipe-card">
       <div class="recipe-header" style="background: #e74c3c;">
           <span class="recipe-number">Ελληνική Συνταγή #21</span>
           <h3>Φάβα με Κολοκύθα (Κρητική)</h3>
   ```

2. **Κόψτε (Cut)** ολόκληρο το `<article class="recipe-card">` για τη συνταγή #21

3. **Κολλήστε (Paste)** στο **Στάδιο 3** μετά τη συνταγή #22 (Σπανακόρυζο)

4. **Αλλάξτε το badge** μέσα στο card από:
   ```html
   <span class="badge safe">✅ 8+ μήνες</span>
   ```
   ΣΕ:
   ```html
   <span class="badge safe">✅ 9+ μήνες</span>
   ```

5. **Προσθέστε** μια προειδοποίηση ΜΕΤΑ το `.cooking-tips` και ΠΡΙΝ το `.badges`:
   ```html
   <div class="age-warning">
       ⚠️ Η φάβα είναι όσπριο με φυτικές ίνες - μόνο από 9+ μήνες
   </div>
   ```

## 🥦 Αλλαγή #2: Προσθήκη Λαχανικών στο Κοτόπουλο με Ρύζι (Συνταγή #20)

### Πρόβλημα
Η συνταγή έχει μόνο **1 μικρό καρότο** για 3 μερίδες. Αυτό δεν προσφέρει αρκετή ποικιλία λαχανικών.

### Λύση
Προσθέστε περισσότερα λαχανικά (καρότα + μπρόκολο) για καλύτερη διατροφική ισορροπία.

### Αναλυτικά Βήματα

1. **Βρείτε** τη συνταγή #20 στο Στάδιο 2:
   ```html
   <!-- Συνταγή 20 -->
   <article class="recipe-card">
       <div class="recipe-header" style="background: #e74c3c;">
           <span class="recipe-number">Ελληνική Συνταγή #20</span>
           <h3>Κοτόπουλο με Ρύζι (Pilaf Style)</h3>
   ```

2. **Αλλάξτε τα υλικά** από:
   ```html
   <li>1 μικρό καρότο (κομμένο σε κυβάκια)</li>
   ```
   
   ΣΕ:
   ```html
   <li>2 μεσαία καρότα (κομμένα σε κυβάκια)</li>
   <li>50 g μπρόκολο (κομμένο σε μικρά ανθάκια)</li>
   ```

3. **Ενημερώστε το Στάδιο 3 (Τοποθέτηση)** από:
   ```html
   <span class="step-content"><strong>Τοποθέτηση:</strong> Βάλτε το κοτόπουλο και το καρότο στο καλάθι ατμού του Philips Avent.</span>
   ```
   
   ΣΕ:
   ```html
   <span class="step-content"><strong>Τοποθέτηση:</strong> Βάλτε το κοτόπουλο, τα καρότα και το μπρόκολο στο καλάθι ατμού του Philips Avent.</span>
   ```

4. **Ενημερώστε το Στάδιο 7 (Συνδυασμός)** από:
   ```html
   <span class="step-content"><strong>Συνδυασμός:</strong> Αφαιρέστε το καλάθι, αδειάστε το νερό. Βάλτε το κοτόπουλο, το καρότο και το μαγειρεμένο ρύζι στο δοχείο.</span>
   ```
   
   ΣΕ:
   ```html
   <span class="step-content"><strong>Συνδυασμός:</strong> Αφαιρέστε το καλάθι, αδειάστε το νερό. Βάλτε το κοτόπουλο, τα καρότα, το μπρόκολο και το μαγειρεμένο ρύζι στο δοχείο.</span>
   ```

5. **Προσθέστε** στο step-detail του Σταδίου 2 (Προετοιμασία Λαχανικών):
   ```html
   <div class="step-detail">🥦 Το μπρόκολο είναι πλούσιο σε βιταμίνη C και άσβεστιο</div>
   ```

## 🧅 Αλλαγή #3: Προειδοποίηση για Κρεμμύδι στο Σπανακόρυζο (Συνταγή #22)

### Πρόβλημα
Η συνταγή αναφέρει κρεμμύδι ως προαιρετικό, αλλά δεν δίνει οδηγίες για την ηλικία. Το κρεμμύδι μπορεί να προκαλέσει αέρια σε μωρά 9 μηνών.

### Λύση
Προσθέστε προειδοποίηση ότι το κρεμμύδι είναι καλύτερα για μωρά 10+ μηνών.

### Αναλυτικά Βήματα

1. **Βρείτε** τη συνταγή #22 στο Στάδιο 3:
   ```html
   <!-- Συνταγή 22 -->
   <article class="recipe-card">
       <div class="recipe-header" style="background: #27ae60;">
           <span class="recipe-number">Ελληνική Συνταγή #22</span>
           <h3>Σπανακόρυζο (Baby Version)</h3>
   ```

2. **Αλλάξτε το υλικό του κρεμμυδιού** από:
   ```html
   <li>1/4 μικρού κρεμμυδιού (μαγειρεμένο - προαιρετικό)</li>
   ```
   
   ΣΕ:
   ```html
   <li>1/4 μικρού κρεμμυδιού (μαγειρεμένο - προαιρετικό, μόνο για 10+ μήνες)</li>
   ```

3. **Ενημερώστε το `.age-warning`** από:
   ```html
   <div class="age-warning">
       ⚠️ ΧΩΡΙΣ άνηθο μέχρι τους 10 μήνες • ΧΩΡΙΣ λεμόνι μέχρι τους 14 μήνες
   </div>
   ```
   
   ΣΕ:
   ```html
   <div class="age-warning">
       ⚠️ ΧΩΡΙΣ άνηθο μέχρι τους 10 μήνες • ΧΩΡΙΣ λεμόνι μέχρι τους 14 μήνες • Κρεμμύδι μόνο για 10+ μήνες (μπορεί να προκαλέσει αέρια σε μικρότερα μωρά)
   </div>
   ```
