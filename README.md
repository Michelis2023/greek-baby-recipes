## 📖 Περιγραφή

**Ελληνικές Συνταγές για Βρέφη** είναι μια διαδικτυακή συλλογή παραδοσιακών ελληνικών συνταγών προσαρμοσμένων για βρέφη, ειδικά σχεδιασμένη για χρήση με τη συσκευή **Philips Avent 4-in-1 Steamer Blender**. Οι συνταγές βασίζονται στη Μεσογειακή διατροφή και την ελληνική λαϊκή σοφία, με αναλυτικές οδηγίες βήμα-βήμα για ασφαλή και υγιεινή διατροφή.

### ✨ Χαρακτηριστικά

- 🍼 **Συνταγές ανά ηλικία**: Οργανωμένες σε στάδια (6+ μήνες, 8+ μήνες, 12+ μήνες)
- 🇬🇷 **Ελληνική παράδοση**: Χρήση ντόπιων υλικών και παραδοσιακών συνδυασμών
- 🔧 **Αναλυτικές οδηγίες**: Βήμα-βήμα χρήση του Philips Avent 4-in-1
- ⚠️ **Οδηγίες ασφαλείας**: Σημαντικές προειδοποιήσεις για κάθε τροφή
- 💚 **Διατροφικά οφέλη**: Πληροφορίες για κάθε συνταγή
- 📱 **Responsive design**: Πλήρως προσαρμοσμένο για κινητά, tablet και desktop

## 🍽️ Διαθέσιμες συνταγές

### Στάδιο 1: Πρώτες Γεύσεις (6+ μήνες)

1. **Πουρές Κολοκύθας Κόκκινης**  
   *Υψηλή σε βήτα-καροτένη, Βιταμίνη A & C*

2. **Πουρές Καρότου με Πατάτα**  
   *Πηγή υδατανθράκων και Καλίου*

### Στάδιο 2: Μαλακές Υφές (7-9 μήνες)

3. **Ελληνικό Γιουβαρλάκια (Baby Version)**  
   *Πλήρης πρωτεΐνη, Σίδηρος και Ψευδάργυρος*

4. **Κοτόπουλο με Ρύζι (Pilaf Style)**  
   *Υψηλή πρωτεΐνη, Βιταμίνες B-complex*

5. **Φάβα με Κολοκύθα (Κρητική)**  
   *Φυτική πρωτεΐνη και Φολικό οξύ*

### Στάδιο 3: Πιο Σταθερές Υφές (9-12 μήνες)

6. **Σπανακόρυζο (Baby Version)**  
   *Σίδηρος, Ασβέστιο, Βιταμίνη K*

7. **Αρακάς με Καρότο και Ρύζι**  
   *Φυτική πρωτεΐνη και φυτικές ίνες*

### Επερχόμενες συνταγές

- Μελιτζανοσαλάτα για Μωρά (10+ μήνες)
- Φασολάδα Απαλή (8+ μήνες)
- Ρεβυθοσούπα με Λαχανικά (9+ μήνες)
- Χυλός με Γιαούρτι και Μέλι (12+ μήνες)

## 🚀 Εγκατάσταση και χρήση

### Προβολή τοπικά

1. **Clone το repository:**
   ```bash
   git clone https://github.com/Michelis2023/greek-baby-recipes.git
   cd greek-baby-recipes
   ```

2. **Άνοιγμα σε browser:**
   - Απλά ανοίξτε το `index.html` σε οποιονδήποτε web browser
   - Ή χρησιμοποιήστε έναν local server:
     ```bash
     python -m http.server 8000
     # Επισκεφτείτε: http://localhost:8000
     ```

### Deploy στο GitHub Pages

Το site είναι ήδη διαθέσιμο μέσω GitHub Pages. Για να ενημερώσετε:

1. Κάντε τις αλλαγές σας στο `index.html`
2. Κάντε commit και push:
   ```bash
   git add .
   git commit -m "Ενημέρωση συνταγών"
   git push origin main
   ```
3. Οι αλλαγές θα εμφανιστούν αυτόματα σε λίγα λεπτά
